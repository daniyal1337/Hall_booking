Oh Puleez – Your Event, Our Perfection
Welcome to the "Oh Puleez" website repository! This project aims to provide users with a seamless experience for event planning and hall booking services. Whether you're hosting a wedding, corporate event, or any other occasion, "Oh Puleez" is dedicated to bringing your vision to life.


The "Oh Puleez" website is an event management platform designed to:

Provide information about the services offered.
Allow users to browse available venues and view their details.
Let users fill in booking forms to inquire or confirm bookings.
Include a contact form for customers to get in touch with the team.
Key functionalities include:

Interactive navigation menus.
Detailed event categories (e.g., Weddings, Corporate Events, etc.).
User-friendly booking forms.
Event gallery showcasing past events.
A "Book Now" section for easy event bookings.
Technologies Used
The website is built using the following technologies:

HTML5: For creating the structure and content of the website.
CSS3: For styling the website elements and layout.
JavaScript: For adding interactivity to the webpage, such as form validation and button interactions.
Bootstrap: For responsive design and layout.
Font Awesome: For icons used in the navigation menu and footer.
WOW.js: For scroll animations.
jQuery: For simplifying DOM manipulation and event handling.
Features
1. Responsive Design
The website is fully responsive, meaning it will adjust to different screen sizes, including mobile devices, tablets, and desktops.

2. Navigation
The header navigation provides access to different sections of the website, including:

Home
Occasions
Capacity
Venue
Gallery

3. Booking Form
Users can fill out a booking form to inquire about services. This includes fields for:

Name

Venue Name
Capacity

Message
4. Footer Section
Includes important links like:

Terms of Service
Privacy Policy
Contact Information (address, phone number, and email)
Social Media Links
5. Dynamic Gallery
A visual gallery of past events to showcase the company’s expertise in event management.

Installation
Follow these steps to run the website locally:

Prerequisites
A modern web browser (Google Chrome, Firefox, Safari, etc.)
An internet connection (for external libraries like Bootstrap and Font Awesome)



How to Use
Navigating the Website: Use the header menu to explore different sections like "Home", "Occasions", "Venue", "Gallery", etc.

Filling the Form: In the "Contact Us" section, fill out the form with your details and submit it for inquiries or bookings. You can also download details through the "Download Details" button.

Booking: Select the appropriate booking options from the available venues and services to confirm your event with us.

Social Media: Follow us on our social media platforms for updates on upcoming events and promotions.
